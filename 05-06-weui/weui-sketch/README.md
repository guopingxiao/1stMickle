### 简介

[WeUI](https://github.com/weui/weui) 是一套由微信官方设计团队为微信 Web 开发量身设计的，同微信原生视觉体验一致的基础样式库。

WeUI-Sketch 是微信官方设计团队设计的 WeUI 全部组件的视觉标准。开发者或设计师可以使用 WeUI-Sketch 进行微信 Web 开发的原型设计、视觉设计等。

### 环境依赖

#### Sketch 版本

不低于 v3.6

#### 字体

- 苹方/ PingFang SC 
- SF UI Text
- 思源黑体  / Noto Sans CJK
- Robot
